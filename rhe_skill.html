Rhetorical Skills (Literary Devices)

譬喻 Figurative language

譬喻是一種通過將一個事物與另一個事物相比較，以創造出更生動的形象的修辭手法。譬諭分為明喻、暗喻、略喻和借喻。

明喻 Simlie

範例：
  1. 他動也不動，彷如石像。
  2. 太陽就像一個大火球。
  3. 我就如一朵花。
  4. 他的臉色像一張灰色的紙。

暗喻 Metaphor

範例：
  1. 你的眼睛是星星。
  2. 用辣宜猛，否則便是昏君庸主。
  3. 歌吹為風，粉汗為雨。
  4. 村舍與樹林是這地盤上的棋子。

略喻 Super Metaphor

範例：
  1. 浮光躍金，靜影沉璧。
  2. 談笑間，檣櫓灰飛煙滅。
  3. 一時人聲鼎沸
  4. 菊，花之隱逸者也。

借喻 Ultra Metaphor

範例：
  1. 天上張著灰色的幔
  2. 亂石崩雲，驚濤裂浪，捲起千堆雪
  3. 縱一葦之所如，凌萬頃之茫然
  4. 松柏後凋於歲寒，雞鳴不已於風雨
  5. 賴蝦蟆想吃天鵝肉。

轉化 Transformation

在描述一件事物時，將它原來的性質轉換成另一種和其本質截然不同、想法一樣的事物，這種修辭法叫做「轉化法」。

擬人 Personification

範例：
  1. 幾點流螢婆娑在我們周圍歌唱。
  2. 屋頂上的雨水滴落下來，理直氣壯的在屋簷下匯成一道水流。
  3. 這是惡習慣戰勝我們的好機會。
  4. 一大幅平原舉起多少甘蔗。
  5. 窗外是一片鳥囀，有時是獨奏，有時是合唱，簡直是一派和諧的交響樂。
  6. 頸部和背部的關節也往往因為低頭太久，不時向我提出嚴重抗議。
  7. 舊的路衰老了，毀壞了。

擬物 Objectification

範例：
  1. 父母倆的感情，已經結成了冰塊。
  2. 爺爺的生命亮起紅燈，全家人擔心死了。
  3. 美人凋謝，令人惋惜。
  4. 你不妨搖曳著一頭「蓬草」，不妨縱容你滿腮的「苔蘚」
  5. 她的快樂已成一縷輕煙，飄的無影無蹤了。

形象化 Concretization

範例：
  1. 我沒有夸父的荒誕，但晚景的溫存卻被我這樣偷嚐了不少
  2. 把回憶風乾，晚年下酒。
  3. 全體同學沐浴在老師的愛心裡。

轉品 Verb Conversion

凡是在語文中，把一個詞彙原來的詞類靈活地轉變為其他詞類的一種修辭技巧，叫做轉品。

範例：
  1. 乃「簪」一花
  2. 妾不「衣」帛
  3. 人不相「非」
  4. 寒不可以「衣」，飢不可以「食」
  5. 吾「從」而師之。
  6. 「親」賢臣，「遠」小人。

誇飾 Hyperbole

誇飾指在語文中，為了表達強烈的情感或鮮明的意象，故意誇大其詞，遠遠超過事實，運用放大或縮小事物的特徵、作用、形象、數量等方式的一種修辭技巧。

範例：
  1. 這杯酒的年齡比我還大。
  2. 千呼萬喚始出來，猶抱琵琶半遮面。
  3. 更妙的是三五月明之夜，天是那樣藍，幾乎透明似的，月亮離山頂，似乎不過幾尺。
  4. 會少離多看兩鬢。萬縷千絲，何況新來病。
  5. 白髮三千丈，緣愁似箇長。
  6. 千山鳥飛絕，萬徑人蹤滅。
  7. 千祿百福，子孫千億。
  8. 在逃去如飛的日子裡，在千門萬戶的世界裡的我，能做些什麼呢？
  9. 子在齊聞韶，三月不知肉味。

排比 Parallelism

排比是一種將同樣的語法結構應用於多個句子中的修辭手法，它可以強調相似或對比之處。

範例：
  1. 她的個性善良，細心，有耐心，是一個值得信賴的人。

反問

反問是一種在提問時暗示答案的修辭手法，它可以引人深思和加強文章的說服力。

範例：
  1. 中文補習對提高成績有幫助嗎？這不言而喻。

對比 Juxtaposition

把兩個相對或相反的事物，或者一個事物的兩個不同面並列，相互對照。對比不分主角配角，相互陪襯。

範例：
    他的臉色像死人一樣蒼白，眼睛像兩個深淵。

借代 Metonymy

不直接說出要說的人和事，而借用相關事物來代替。借代最常見的類別有用專名代通名、用事物的特徵代事物、用具體事物代抽象事物、用個別事物代一般事物等。

範例：
  1. 南國烽煙正十年
  2. 一隊藍帽子到年宵市場外維持秩序。
  3. 槍桿子裏出政權。
  4. 三個臭皮匠，勝過一個諸葛亮。

襯托

把兩個 相關或相對的事物(意思) 或 同一事物的兩個不同方面放在一起。利用類似事物的正反面、有差別的事物作陪襯，突出主要事物。

正襯

範例：
  1. 「桃花潭水深千尺，不及汪倫送我情」

反襯

  1. 「吳天寶人小，氣量可大，看出姚大嬸氣色不善，也不介意」

對偶 Antithesis

把兩個結構相同或相似、意思相關、字數相等的短語或句子對稱地排列在一起的修辭

反復

為了表達的需要，例如：強調某個意思或者情感，而刻意重複某個句子或者詞語。

頂真 Anadiplosis

頂真 ，亦叫聯珠 。把上句末尾的詞語作為下句開頭的詞語，一句句首尾相連的修辭方式。

設問 Rhetoric question

提問

凡是在語文中，提起下文而自問自答的一種修辭技巧。

範例：
  1. 什麼是路？就是從沒有路的地方踐踏出來的，從只有荊棘的地方關闢出來的。
  2. 甚麼叫自律？ 自律就是自己管束自己的行為。

激問

凡是在語文中，激發本意，問而不答，而答案在問題的反面的一種修辭技巧。

懸問

凡是在語文中，作者內心確實存有疑惑，而刻意將此疑惑懸示出來詢問讀者的一種修辭技巧。

疊字

把字義相同的漢字重疊一齊使用組成詞語。

反語 Verbal irony

顧名思義，即「說反話」，是故意令表面的意思和實際要表達的意思相反的修辭手法。

雙關 Double entendre

即一語雙關，利用詞語的多意性、發音等的特點，在使用一個詞語的同時可以同時表達到兩種事物或意義。
